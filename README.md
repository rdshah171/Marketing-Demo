# Agentic Campaign Optimization Engine

A Streamlit workshop app that demonstrates how AI agents can analyze marketing campaign data, identify underperforming segments, rewrite ad creative, and reallocate budget.

## Project Structure

- `app.py` - main Streamlit application shell with page navigation.
- `data/data_generator.py` - dummy marketing dataset generation and Excel export.
- `README.md` - project overview and run instructions.

## Run Locally

1. Install dependencies: `pip install streamlit pandas plotly openpyxl`
2. Launch the app: `streamlit run app.py`

## Features

- Campaign dashboard with interactive charts
- Anomaly detection for underperforming segments
- Agentic creative optimization simulation
- Budget reallocation recommendations
